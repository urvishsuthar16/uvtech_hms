Login

1. Access the Login Page: To log in to the system, go to https://spl.uvtechglobal.com/login. You will be taken to the login page where you need to enter your credentials.

2. Enter Credentials: Type your username and password, then click "Login". Once authenticated, you will be redirected to the SPL Manager home page.

3. Accessing Shortcuts: From the home page, you will have access to all shortcuts, such as task management, inventory, and more.

User and Employee Creation

Navigate to the Users Section

1. Go to Users Section: To add new users, navigate to the Users section in the system. This is where you manage user accounts.

Create Two Types of Users
  • Manager: Assign the SPL Manager role. This user will have access to advanced system features.
  • Staff: Assign both SPL Staff and Employee roles. Staff members have specific access permissions suited for day-to-day tasks.
Assign Module Permissions

2. Module Permissions: Both SPL Manager and SPL Staff roles should be granted access to necessary modules like HMS and Desk. These permissions allow users to perform key functions based on their role.

Location (Project) and Task Template Creation

Navigate to the Projects Section

1. Access Projects: Go to the Projects section to create and manage locations or specific projects. This is the primary area where new projects are created.

Assign Users to the Project

2. User Assignment: While creating a project, make sure to assign the relevant users. Only the users assigned to the project will be eligible to receive tasks related to that project.

3. Single Project Assignment: Keep in mind that a user can only be assigned to one project at a time. This ensures clarity in task distribution and tracking.

Task Template Creation

Navigate to the Task Templates Section

1. Create Templates: Visit the Task Templates section to create task templates that can be reused. Templates help streamline task creation across different projects.

Enable 'Is Template'

2. Mark as Template: While creating a task, be sure to enable the 'Is Template' checkbox. This ensures that the task is reusable and available for future projects.

Inventory Management

Fetch Latest Supplier and Price Information

1. Supplier and Price Updates: When managing inventory, click the "Assign Supplier" button to automatically fetch the latest supplier and price information. This pulls the most current pricing details for items in your inventory from external price lists.

Automatic Price Updates

2. Dynamic Pricing: As users select or update a supplier, the system dynamically fetches and applies the latest price. This real-time update ensures pricing accuracy without manual effort.

Filtering and Copying Data

3. Filter and Copy: Use the inventory filter to locate specific items, then copy filtered data such as item code, quantity, and price with a single click. This feature aids in easy data export or sharing.

Note: SPL Managers can send email notifications to all suppliers by clicking the "Send Mail" button, streamlining supplier communication.

Attendance Management

Mark Manual Attendance

1. Manual Attendance: To mark attendance manually, navigate to the Attendance section via the shortcut on the home page. From here, you can record attendance for all employees under your supervision.

Generate Weekly Timesheet

2. Generate Timesheet: Go to the Timesheet Report shortcut. This opens a page displaying attendance records for the past week. Adjust the date range to see different time periods or employee-specific records.

View and Export Timesheet Report

3. Export Reports: On the report page, view detailed employee attendance data. Export the timesheet for further analysis or sharing.

Change User Password

Change Password

1. Go to User Doctype List: Navigate to the User Doctype list by searching for "User" in the search bar or through the User section in the setup menu.

Select User

2. Select User: In the User list, find and click on the specific user for whom you want to change the password.

Open the Settings Tab

3. Go to the Settings Tab: Once you're inside the user's profile, click on the "Settings" tab to open it.

Change Password

4. Enter New Password: In the "Change Password" section, input the new password. You can click the eye icon to view the entered password for accuracy.

Login

  • URL: https://spl.uvtechglobal.com/login
  • Enter username and password.
  • Upon login, access the home page for Task, Inventory, Attendance, and Timesheet Report shortcuts.

Attendance

  • Access Attendance Page: Click on the Attendance shortcut from the home page.
  • Select Shift: If no default shift is assigned, choose one from the dialog prompt.
  • Start Attendance Timer: Click the "Start Time" button to begin the timer.
  • Stop Attendance Timer: Click the "End Time" button to stop the timer.
  • View Employee Timesheet Report: Select an employee, enter the start and end dates, and view their timesheet report.

Task Management

  • Navigate to Tasks: Click on the Tasks shortcut from the home page.
  • Generate Tasks: Click the Generate button and select a shift from the popup.
  • Task Creation: Tasks are automatically created and assigned after shift selection.
  • Complete Tasks: Click the Complete button to upload a file. Completed tasks will display a line-through effect.
  • Shift Change Handling: Incomplete tasks from the previous shift are removed when the user changes shifts, and new tasks are assigned for the new shift.

Inventory Management

  • Navigate to Inventory: Click on the Inventory shortcut from the home page.
  • View Default Items: The inventory page displays default items.
  • Update Item Details: Select an item and update its Price and Quantity.
  • Save Changes: Click the Save button to apply changes.
